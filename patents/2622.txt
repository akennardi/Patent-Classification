Abstract
A fencing clamp has one or more hexagonal cuts to allow the fencing clamp to be
used as a fencing spanner. The positioning of the hexagonal cut allows for 360
degree rotation, allowing for the nut to be tightened without the need for
additional tools.

Fencing Clamp with Integrated Spanner
The innovation is a modification to an existing Fencing Clamp.
The innovation entails one or more hexagonal cuts into the existing fencing
clamp and allow the fencing clamp 'itself' to be used as a fencing spanner.
The positioning of the hexagonal cut allows for 3600 rotation, allowing for the
nut to be tightened.
